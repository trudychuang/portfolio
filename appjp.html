Concept
以旅遊為主題，學習旅日的基本日文
app 系統以強調口說會話做設計，不僅學會日語，還要能實際開口說。
User Pain Point & Solution
Functional Map
Style
UI Overviews - 學習五步驟
學習五步驟分為：學習新課程、跟讀練習、句子填空、對話練習及綜合選擇題，
透過完成這五步驟學習一個完整的課程，可以更加深學習記憶。
UI Overviews - 學習三寶
在學習上提供幫助，分為學習檔案，紀錄學習歷程、五十音表，加強日文基礎、生存指南，收錄旅遊常用單子及句子。
Conclusion
希遊日記提供什麼學習上的幫助？
・旅遊導向學習，強調口說能力，在旅遊時絕對需要並且派上用場。
・系統化學習語言，讓初學者不會迷失在尋找教材中，可以更有效率學習。
・學習三寶的五十音表、生存指南實用性高，是學習日語必備教材，使用時快速方便。
